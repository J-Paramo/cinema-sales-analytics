# Cinema Sales Analytics

Proyecto de inteligencia de negocios enfocado en el análisis de operaciones cinematográficos para la toma de decisiones basadas en datos.

## Resumen del proyecto

El objetivo es analizar el desempeño operativo de distintos complejos cinematográficos, incluyendo ventas de boletos, niveles de ocupación y comportamiento por horario, con el fin de identificar patrones, ineficiencias y oportunidades de mejora en los ingresos.

## Problema de negocio

Las cadenas de cines necesitan entender:

- ¿Qué complejos generan mayores ingresos?
- ¿Qué horarios tienen mayor ocupación?
- ¿Cómo influye el precio del boleto en las ventas?
- ¿Qué días o meses tienen mejor desempeño?
- ¿Qué tan eficiente es la ocupación de las salas?

## Objetivo

Desarrollar un dashboard interactivo en Power BI que permita analizar el desempeño operativo de una cadena de cines, facilitando la toma de decisiones.
